# 🌿 Jharkhand Tourism Website

> **Explore the Land of Forests** — Waterfalls, temples, wildlife, and valleys. A responsive, accessible, and performance-optimized tourism portal for the state of Jharkhand, India.

---

## 📌 Overview

The **Jharkhand Tourism Website** is a front-end web application designed to help travelers discover and plan trips to the scenic destinations of Jharkhand. It provides curated information on tourist spots — including waterfalls, wildlife sanctuaries, temples, hill stations, and valleys — along with travel routes, nearby city distances, and stay facilities, all in one place.

This project was built as part of an **internship assignment**, with a focus on responsiveness, accessibility, and web performance best practices.

🔗 **Live Demo:** [https://jharkhand-tourism-website.vercel.app/](https://jharkhand-tourism-website.vercel.app/)

---

## ✨ Features

- 🏞️ **Curated Tourist Spots** — Explore top destinations across Jharkhand including waterfalls, temples, nature valleys, and wildlife reserves.
- 🔍 **Search Functionality** — Quickly find places using the built-in search bar.
- 📍 **Google Maps Integration** — View each destination directly on Google Maps.
- 🗺️ **Travel Information** — Nearby cities, distances, and route suggestions for each spot.
- 🏨 **Stay Facilities** — Information on accommodation options near tourist locations.
- 📸 **Rich Imagery** — High-quality visuals for every destination to inspire travel.
- 🎥 **Video Background** — Immersive hero section with a scenic video of Jharkhand.
- 📱 **Fully Responsive** — Optimized for mobile, tablet, and desktop screens.
- ♿ **Accessible** — Built with accessibility in mind (semantic HTML, ARIA roles, keyboard navigation).
- ⚡ **Performance Optimized** — Lazy-loaded images, proper aspect ratios, and minimal Cumulative Layout Shift (CLS).
- 📢 **Tourist Guide Recruitment Banner** — Highlights open applications for qualified tourist guides across Jharkhand.

## 🚀 Getting Started

### Prerequisites

- A modern web browser (Chrome, Firefox, Edge, Safari)
- [Git](https://git-scm.com/) installed on your machine
- (Optional) [VS Code](https://code.visualstudio.com/) with Live Server extension

### Installation

1. **Clone the repository:**
   ```bash
   git clone https://github.com/your-username/jharkhand-tourism-website.git
   ```

2. **Navigate into the project directory:**
   ```bash
   cd jharkhand-tourism-website
   ```

3. **Open in your browser:**
   - Simply open `index.html` in a browser, **or**
   - Use the Live Server extension in VS Code for hot-reloading during development.

---

## 🌐 Deployment

This project is deployed on **[Vercel](https://vercel.com/)**. To deploy your own version:

1. Push your code to a GitHub repository.
2. Log in to [vercel.com](https://vercel.com/) and click **"New Project"**.
3. Import your GitHub repository.
4. Vercel will auto-detect the static site and deploy it instantly.
